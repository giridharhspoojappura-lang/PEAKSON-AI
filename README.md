PEAKSON AI is a Flask-based intelligent assistant . It supports both text and voice interaction, analyzes uploaded images and PDF documents, generates AI-powered responses, and maintains conversation logs for future reference.

## Features

* Text Chat
* Voice Input (Speech Recognition)
* Voice Output (Text-to-Speech)
* Image Analysis
* PDF Question Answering
* File Upload Support
* Conversation Logging with Timestamps
* Dynamic Greetings
* Multiple Voice Modes
* Modern JARVIS-Inspired User Interface

## Run Locally

### Install Dependencies

```bash
pip install -r requirements.txt
```

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:5000
```
